# Voice Resemblance Detection using Machine Learning

## Overview
This project implements a basic machine learning system to analyze
voice similarity and speaker resemblance between audio samples.

## Methodology
- Audio preprocessing
- Feature extraction using Python and NumPy
- Similarity comparison using machine learning techniques
- Exploration of transformer-based models (basic)

## Tools & Technologies
- Python
- NumPy
- PyTorch
- Hugging Face Transformers

## Status
This is a learning-focused project and serves as a foundation
for future improvements.
